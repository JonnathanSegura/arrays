# arrays
Programas en Java que implementan ejercicios clásicos de lógica, probabilidad y generación de secuencias. Incluye simulaciones como el paradoja del cumpleaños, generación de tableros de Minesweeper, muestreo con distribuciones discretas, y construcción de la secuencia Thue–Morse, siguiendo un enfoque educativo y algorítmico.
